State Populations

Census Data

Population per area was calculated for each year based on population estimates from the census dataset.

Note: I did not research what formula determined the census estimate, so there is an undetermined margin for error here. This is the best available resource for local populations at this time.

State Area: Square Miles of Land

State Symbol USA

There is one set of data for each state, regardless of the year.


Note: This is a measurement of land, not residential land, so a state with large areas of uninhabitable land (e.g. national parks) may have more densely populated cities, but appear lower on this list.
